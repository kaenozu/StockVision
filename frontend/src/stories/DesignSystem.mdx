## タイポグラフィ

### フォント

プライマリフォント: **Inter**
- ウェブに最適化された可読性の高いフォント
- 日本語との組み合わせを考慮

フォールバック: **"Segoe UI", "Roboto", "Helvetica Neue", sans-serif**

コードフォント: **"Fira Code", "SF Mono", "Monaco", "Cascadia Code", "Roboto Mono", monospace**

### サイズスケール

| サイズ | クラス | 使用用途 |
|--------|--------|----------|
| 12px | text-xs | キャプション、補足テキスト |
| 14px | text-sm | 小さなテキスト、ラベル |
| 16px | text-base | 本文テキスト（デフォルト） |
| 18px | text-lg | 強調テキスト |
| 20px | text-xl | 小見出し |
| 24px | text-2xl | 中見出し |
| 30px | text-3xl | 大見出し |
| 36px | text-4xl | ヒーロータイトル |

## 影とエレベーション

| レベル | クラス | 使用用途 |
|--------|--------|----------|
| 1 | shadow-sm | カード、軽い区切り |
| 2 | shadow | 一般的なカード |
| 3 | shadow-md | 浮いているカード |
| 4 | shadow-lg | モーダル、ドロップダウン |
| 5 | shadow-xl | オーバーレイ、ポップオーバー |

## アクセシビリティ

### カラーコントラスト比

- AAA準拠: 7:1以上（小さなテキスト）
- AA準拠: 4.5:1以上（一般的なテキスト）
- AA準拠（大きなテキスト）: 3:1以上

### フォーカス状態

すべてのインタラクティブ要素に視覚的なフォーカス表示を提供：

```css
focus:outline-none focus:ring-2 focus:ring-blue-500 focus:ring-offset-2
```

### 株価色の配慮

色覚障害者にも配慮し、色だけでなく記号（+/-）や方向（↑↓）でも情報を伝達。
