# StockVision UI コンポーネントライブラリ

StockVision アプリケーション用のReact UIコンポーネントライブラリです。
このStorybookには、株式取引アプリケーションで使用される各種コンポーネントが含まれています。

<div className="subheading">主な機能</div>

- **アクセシビリティ対応**: すべてのコンポーネントがWCAGガイドラインに準拠
- **レスポンシブデザイン**: モバイルファーストのアプローチ
- **国際化対応**: 日本語UI/UXを重視した設計
- **テーマサポート**: ライト/ダークテーマの切り替え
- **TypeScript**: 完全なTypeScriptサポート
- **テスト可能**: Storybook Testing libraryによるインタラクションテスト

<div className="subheading">使用技術</div>

- React 19
- TypeScript
- Tailwind CSS
- Chart.js (チャート表示)
- Storybook 8
- Vite (ビルドツール)
