# StockVision ドキュメント

ようこそ、StockVision ドキュメントへ！

StockVision は、React + TypeScript (フロントエンド) と FastAPI (バックエンド) で構築された株価監視Webアプリケーションです。Yahoo Finance APIとの連携により、リアルタイムな株価情報を提供します。
